\chapter*{Résumé}

\noindent Cette note méthodologique décrit le fonctionnement de \txl, le modèle de micro-simulation de l'ensemble du système socio-fiscale français sur l'ensemble du cycle de vie développé à l'Institut des politiques publiques (IPP). Le modèle \txl \ est un modèle de micro-simulation dynamique non spécifique qui simule pour un échantillon représentatif de la population française les prestations dans leur définition la plus large et les prélèvements, pour l'instant en mettant de côté les taxes sur les entreprises. Des réformes de la législation peuvent aussi être appliquées.

% Le développement du modèle s'inscrit dans le cadre d'une collaboration scientifique avec l'Insee, \PENSIPP s'appuyant largement sur le modèle développé par la division \og RPS \fg de l'Insee, le modèle Destinie 2. A partir de cette structure, \PENSIPP opère trois modifications sensibles: (i) une modification des données sources qui reposent sur un appariement statistique entre l'Échantillon interrégime des cotisants (EIC) et l'enquête Patrimoine pour les données de carrière rétrospectives; (ii) une modélisation des carrières en prospectif qui s'appuie sur les données de l'EIC; (iii) une modélisation du système de retraite sous le logiciel libre R.
